        About + Guide

        The Log is a log and time-tracking system created by Josh Avanier. Its primary purpose is to provide insight to how one's time is spent — insight that can then be used to optimise personal productivity and better manage time.

        Usage

        Use the console to enter commands:

        To start a log: start "Sector" "Project Name" "A description of what you're doing"

        To end a log: end
